## Decisiones de Diseño e IA
* **Paleta Corporativa:** Se extrajeron los códigos hexadecimales del logo oficial (#2E1A47 y #F2B705) para asegurar coherencia visual.
* **Mascota Usagi:** Se implementó como un elemento de identidad visual secundaria mediante animaciones CSS (`@keyframes`) para mejorar la experiencia de usuario (UX),
* **Adaptación de Bocetos:** La IA interpretó los bocetos a mano para generar una cuadrícula responsiva de Bootstrap (`row/col`) que respeta la posición de los módulos (Perfil a la izquierda, tablas inferiores, etc.).
* **Semántica:** Se validó el uso de etiquetas `header`, `nav`, `main`, `section` y `footer` para cumplir con el puntaje máximo de la rúbrica.
